# Next typescript


## 🏃‍♂️ Simple start
1- clone repo
2- run ```npm i```
3- fill envs with your data

```
DATABASE_URL=""

AUTH_SECRET=""

GITHUB_CLIENT_ID=''
GITHUB_CLIENT_SECRET=''

GOOGLE_CLIENT_ID=''
GOOGLE_CLIENT_SECRET=''


RESEND_API_KEY=''
```
4- run command ```npm run dev```
